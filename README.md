# Financial Performance Analysis Dashboard using Excel

## Project Overview

Developed an Excel-based financial dashboard to analyze profitability trends across markets and fiscal years.

The dashboard helps stakeholders monitor:

- Net Sales
- Cost of Goods Sold (COGS)
- Gross Margin
- Gross Margin %
- Year-over-Year Growth
- Market Performance

---

## Business Problem

Business leaders needed a consolidated reporting solution to:

- Monitor financial performance across markets
- Evaluate profitability trends
- Identify high-performing regions
- Support strategic decision-making

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Power Query
- Pivot Charts
- Financial KPI Analysis

## Key Insights

- Strong revenue growth observed across fiscal years.
- Gross Margin % declined despite increasing revenue.
- High revenue growth does not necessarily translate to improved profitability.

---

## Business Recommendations

- Investigate drivers behind declining margins.
- Improve procurement efficiency.
- Focus expansion on high-margin markets.
- Monitor profitability alongside revenue growth.
